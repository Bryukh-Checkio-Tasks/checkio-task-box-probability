Pendant que Stephen pilote un cargo, Nicola et Sophia ont inventé un nouveau jeu avec les boîtes.

Pour commencer le jeu, ils placent plusieurs perles noires et blanches dans l'une des boîtes. Chaque robot peut jouer N coups, à la suite de quoi on retourne à la configuration initiale pour le joueur suivant. À chaque coup, le robot prend une perle dans la boîte, et replace un perle de la couleur contraire dans la boîte. Le gagnant est celui qui tire la perle blanche au Nième coup (ou +1 point s'ils font plusieurs parties).

Nos robots n'aiment pas l'indétermination, et veulent connaître la probabilité d'avoir une perle blanche au Nième coup. La probabilité est une valeur entre 0 (0% de chances si l'événement n'arrivera pas) et 1 (100% de chances si l'événement arrivera à coup sûr). Le résultat est un float entre 0 et 1 avec une précision de deux chiffres après la virgule (±0.01).

On vous donne un ensemble initial de perles sous la forme d'une chaîne de caractères constituée de "b" (black = noir) et de "w" (white = blanc), et le nombre de coups (N). L'ordre des perles n'a pas d'importance.

Input: La distribution initiale de perles, comme chaîne de caractères, et le nombre de coups (int).

Output: La probabilité d'obtenir une perle blanche (float).

Exemple: 

checkio('bbw', 3) == 0.48
checkio('wwb', 3) == 0.52
checkio('www', 3) == 0.56
checkio('bbbb', 1) == 0
checkio('wwbb', 4) == 0.5
checkio('bwbwbwb', 5) == 0.48

Application: Cette mission vous fait manipuler des probabilités élémentaires. Amusant : indépendamment du contenu initial de la boîte, lorsque le nombre de coups augmente, la probabilité approche 0.5 !

Préconditions: 0 < N ≤ 20
0 < |pearls| ≤ 20
